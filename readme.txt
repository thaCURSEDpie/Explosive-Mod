Type:           game modification (GTA:IV)
Language:       C#
Description:
    A mod for GTA:IV which adds a free-flying camera. Relies on the .Net Scripthook by HazardX
    
More info:
    Nothing special. Most important thing is moving the player around with the camera, otherwise the game will not load the region.
    
    Configuration file located in 'etc'.
    
Compiling:
    Not needed, but if you really have to: grab "ScriptHookDotNet.dll" here: http://www.gtaforums.com/index.php?showtopic=392325 and have some fun.
